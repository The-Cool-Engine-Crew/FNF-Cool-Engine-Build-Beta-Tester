# 🎵 FNF Cool Engine --- Beta Build

Welcome to the **FNF Cool Engine Beta Build**, an experimental version
of the engine focused on **testing, experimenting, and improving new
features and menus** before official releases.

This repository provides a playable and moddable build designed for
**beta testers, programmers, and mod creators** who want to explore the
engine early and help shape its development.

------------------------------------------------------------------------

## 🚧 About This Project

**FNF Cool Engine** is a custom engine for **Friday Night Funkin'** that
introduces new systems, improved menus, and expanded customization
possibilities.

This beta build allows the community to:

-   🎮 Play and test new features
-   🧪 Try experimental menu systems
-   🛠 Modify and program new content
-   🧩 Create and test mods
-   🐞 Report bugs and suggest improvements

Expect unfinished features, occasional bugs, and rapid changes ---
that's part of the beta experience!

------------------------------------------------------------------------

## ✨ Features (Beta)

-   ✅ New redesigned menu system
-   ✅ Experimental UI elements
-   ✅ Modding-friendly structure
-   ✅ Open for customization and scripting
-   ✅ Active development

More features will be added continuously as the engine evolves.
